# Pandoc - 文档转换核心引擎
# ----------------------------------------------------------------------------
# Pandoc 是一个独立的命令行工具，无法通过 pip 安装
# 必须单独安装后才能使用 docx_to_markdown.py 模块
#
# 安装方法：
#
#   Windows (推荐使用 winget，Windows 10/11 自带):
#     winget install --id=JohnMacFarlane.Pandoc -e
#
#   Windows (备选方案 - Chocolatey，需要管理员权限):
#     choco install pandoc
#
#   macOS (使用 Homebrew):
#     brew install pandoc
#
#   Linux (Ubuntu/Debian):
#     sudo apt-get update && sudo apt-get install -y pandoc
#
#   或从官网下载: https://pandoc.org/installing.html
#
# 验证安装:
#   pandoc --version
#
# ============================================================================

# ============================================================================
# 安装指南
# ============================================================================

# 方法1: 使用自动安装脚本 (强烈推荐)

# 方法2: 手动安装
# ----------------------------------------------------------------------------
#   # 安装 Python 包

#   winget install --id=JohnMacFarlane.Pandoc -e  # Windows 10/11
#   choco install pandoc                          # Windows (需管理员)
#   brew install pandoc                           # macOS
#   sudo apt-get install pandoc                   # Linux
#
# 方法3: 最小安装 (仅命令行)
