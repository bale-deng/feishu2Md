# 📋 版本历史

## 版本命名规则

采用语义化版本号：`主版本号.次版本号.修订号`

- **主版本号**：重大架构变更或不兼容的 API 修改
- **次版本号**：新功能添加，向下兼容
- **修订号**：Bug 修复和小改进

---

## 🚀 V1.0.03 (2025-11-13)

### 🐛 Bug 修复

- **修复 exe 环境依赖安装问题**
  - 修复 `feishu2md.exe` 在无源代码环境下无法安装依赖的问题
  - 将依赖安装逻辑完全集成到 `feishu2md.py` 中
  - 移除对外部 `install_dependencies.py` 的依赖
  - exe 版本现在可以完全独立运行，自动安装所有依赖

### 🔧 优化改进

- **依赖安装流程优化**
  - 统一的 GUI 安装进度显示
  - 更完善的错误提示和解决方案
  - 自动验证安装结果
  - 支持 winget 和 Chocolatey 双安装方式

### 📚 架构说明

- **双层保障机制**
  - `install_dependencies.py`：独立工具，适合有源代码的开发者
  - `feishu2md.py` 内置检测：自动化安装，适合 exe 用户
  - 确保所有场景都能顺利安装依赖

---

## 🎯 V1.1.0.02 (2025-11-13) ---> 未发布

### 🔧 优化改进

- **Pandoc 安装逻辑优化**
  - Windows 平台优先推荐使用 winget 安装（无需管理员权限）
  - Chocolatey 作为备选方案（需要管理员权限）
  - 添加管理员权限检测和自动安装功能
  - `feishu2md.exe` 支持启动时自动安装 Pandoc
    - 使用 winget：无需管理员权限
    - 使用 choco：需要管理员权限
  - 优化 `install_dependencies.py` 的错误提示和引导

### 📚 文档更新

- 更新 Pandoc 安装说明，强调权限要求
- 简化安装流程说明，减少用户困惑
- 添加多处管理员权限相关提示
- 优先推荐 winget 进行 Pandoc 安装

## ✨ V1.0.01 (2025-11)

### 新增功能

- **全新深色主题 GUI 界面**
  - 基于 tkinter 的现代化图形界面
  - 深色主题 + 粉色按钮 + 绿色进度条
  - 1200×850 分辨率，适配主流显示器

- **飞书表格自动转换功能**
  - 智能识别飞书文本表格格式
  - 自动转换为标准 Markdown 表格
  - 保护已有的 Markdown 表格不被破坏

- **智能表格格式保护**
  - 检测并保护 Markdown 表格（`|` 分隔符）
  - 检测并保护飞书文本表格
  - 智能处理反斜杠（仅在非表格行移除）

- **代码块与表格智能区分**
  - 纯连续短横线识别为代码块
  - 空格分隔短横线识别为表格
  - 支持 30+ 种编程语言检测

- **依赖自动安装工具**
  - `install_dependencies.py` 自动检测和安装依赖
  - 支持交互式安装和静默安装模式
  - 跨平台支持（Windows/macOS/Linux）

### Bug 修复

- 修复图片路径问题
- 修复表格转换时的格式错误
- 修复代码块识别误判
- 修复 HTML 标签清理不完整
- 修复文件编码问题

### 文档改进

- 完善 README.md 文档
- 添加详细的安装指南
- 提供完整的使用示例
- 增加故障排除章节
